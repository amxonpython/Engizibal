# Engizibal

Консольное приложение на Java с набором утилит для повседневного использования.  
Проект пишется в учебных целях.

---

## Возможности

| Команда | Описание |
|---|---|
| `calculator` | Простой калькулятор (сложение, вычитание, умножение, деление) |
| `saved text` | Мини-блокнот: сохранить, посмотреть или удалить текст |
| `echo <текст>` | Вывести введённый текст обратно |
| `date` | Показать текущую дату |
| `time` | Показать текущее время |
| `name` | Сохранить и отобразить ваше имя |
| `help` | Список всех команд |
| `exit` | Выйти из программы |

---

## Структура проекта

```
├── Main.java          # Точка входа, главный цикл
├── clases/
│   ├── classes.java   # Дата и время
│   ├── commands.java  # Обработка команд
│   ├── data.java      # Хранение данных и список команд
│   ├── Errors.java    # Сообщения об ошибках
│   └── test_class.java # Калькулятор
```

---

## Запуск

для запуска просто запустите файл **Main.java** в любом компиляторе кода
